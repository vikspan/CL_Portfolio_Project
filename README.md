### Prerequisites
Make sure you have:
- Node.js
- npm or yarn

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/vikspan/portfolio
cd portfolio
npm install
# or
yarn install
npm run dev
```

## Run locally

```bash
npm run dev
```

Then open your browser and visit: http://localhost:3001
